# Vrinda Store Data Analysis

## Project Overview
This project analyzes the annual sales data of Vrinda Store for the year 2022. It includes insights into:
- Monthly sales and order trends.
- Gender-based sales distribution (Men vs. Women).
- Order statuses (Delivered, Cancelled, Refunded, etc.).
- Sales distribution by states.

## Inspiration
This project was created by following a tutorial from [YouTube Channel Name](YouTube Link).  
Special thanks to the creator for providing guidance and inspiration.

## Key Features
- Order vs Sales: Monthly breakdown of the number of orders and sales amount.
- Gender-Based Analysis: Comparison of sales generated by male and female customers.
- Order Status: Distribution of orders by their statuses (e.g., Delivered, Cancelled).
- State-Wise Sales: Sales data categorized by geographical location.

## Tools Used
- Microsoft Excel:
  - Data cleaning and organization.
  - Visualizations for better insights.
- Data Analysis Techniques:
  - Trend analysis.
  - Segmentation and performance tracking.

## How to Use
1. Open the Excel file: `Vrinda Store Data Analysis.xlsx`.
2. Explore the sheets:
   - Report: Summary of the data insights.
   - Order VS Sales: Monthly sales trends.
   - Men vs Women: Gender-based sales distribution.
   - Order Status: Analysis of order completion status.
   - States: Sales by geographical location.
